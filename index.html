Projects & Labs

AWS ECS Fargate Microservices Infrastructure

End-to-end IaC project that provisions a secure, cost-optimized microservices stack on AWS ECS Fargate using Terraform. The architecture includes a private VPC with endpoints (ECR, Secrets Manager, CloudWatch Logs), RDS MySQL, AWS WAF, GuardDuty, VPC Flow Logs, SNS alerts, and no NAT Gateway to reduce attack surface and cost. A CI/CD pipeline (GitHub → CodeBuild → CodePipeline → ECS) automatically builds Docker images, pushes them to ECR, and triggers zero-downtime rolling updates for the ECS service on every code push.

Vulnerability Assessment – WordPress

Analyzed security weaknesses in a WordPress-based website and discovered multiple vulnerabilities including Clickjacking (UI Redressing) and user enumeration via the REST API combined with XML-RPC multicall brute-force attacks. Conducted proof-of-concept (PoC) demonstrations to validate the issues, including a clickjacking phishing page and a Python script leveraging XML-RPC multicall to perform automated credential testing. The project demonstrates how misconfigured HTTP security headers and unrestricted WordPress features can expose administrative accounts to phishing and automated credential attacks.
